This is a simple form that will invoke a unix shell script  that will list the contents
of any websys directory. Right click on the link below, and then
click "Save As"  "lsstud.cgi". See below...

lsstud.cgi -- (Right Click this link and SAVE AS lsstud.cgi to get a copy)

 

Right click this link and SAVE AS lsstud.html for the form code

 Move (sftp or Dreamweaver) the lsstud.cgi program and the lsstud.html program to your websys directory

 and make sure they work (point your browser at http://vleda..stern.nyu.edu/~youruserid/websys/lsstud.html). Remember to chmod o+r to the lsstud.html program

 and chmod o+rx to the lsstud.cgi program before you try them, so the web server process has the correct access rights.
